## Getting Started

Follow these steps to run source code.

### Prerequisites

Node Version

```shell

$ batch> node -v
v18.18.0
$ batch> npm -v
9.8.1

```

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/phananhtuan09/EZ-backend.git
   ```

2. Pull latest code

   ```sh
   git pull origin develop
   ```

3. Install NPM packages

   ```sh
   npm install
   ```

4. Create file `.env` in the root folder
   ```js
   DATABASE_URL = "ENTER YOUR DATABASE MONGODB URL";
   ```
